Create a new file `src/actions/project-actions.ts` with a server action `migrateProjectPath`:

```typescript
"use server";

import { rename, mkdir, access, readdir } from "fs/promises";
import { existsSync } from "fs";
import path from "path";
import { execFileSync } from "child_process";
import { db } from "@/lib/db";
import { revalidatePath } from "next/cache";
```

**Function signature:**
```typescript
export async function migrateProjectPath(
  projectId: string,
  targetPath: string
): Promise<{ success: true } | { success: false; error: string }>
```

**Implementation steps:**

1. **Validate inputs**: projectId must be non-empty string, targetPath must be non-empty absolute path. Return `{ success: false, error: "..." }` for invalid inputs.

2. **Load project from DB** (`db.project.findUnique({ where: { id: projectId } })`). Return error if not found or no localPath.

3. **Pre-flight: running executions** — Query `db.taskExecution.findMany({ where: { task: { projectId }, status: "RUNNING" } })`. If any exist, return `{ success: false, error: "该项目有正在运行的任务执行，请先停止所有执行" }`.

4. **Pre-flight: active PTY sessions** — Dynamically import `getSession` from `@/lib/pty/session-store`. Get all tasks for the project (`db.task.findMany({ where: { projectId }, select: { id: true } })`). Check if any task has an active PTY session (`getSession(taskId) !== undefined`). If any exist, return `{ success: false, error: "该项目有活跃的终端会话，请先关闭" }`.

5. **Pre-flight: existing worktrees** — Check if `.worktrees` directory exists at source path and has any entries via `readdir`. If worktrees exist, return `{ success: false, error: "该项目有活跃的 Git Worktree，请先清理" }`.

6. **Pre-flight: target path** — If targetPath already exists and is non-empty, return `{ success: false, error: "目标路径已存在且不为空" }`. If targetPath === sourcePath, return `{ success: false, error: "源路径和目标路径相同" }`.

7. **Create parent directory**: `await mkdir(path.dirname(targetPath), { recursive: true })`.

8. **Atomic rename**: `await rename(sourcePath, targetPath)`. Wrap in try-catch:
   - If error.code === "EXDEV", return `{ success: false, error: "源路径和目标路径不在同一文件系统，不支持跨设备迁移" }`.
   - For other errors, return `{ success: false, error: error.message }`.

9. **Update database**: `await db.project.update({ where: { id: projectId }, data: { localPath: targetPath } })`.

10. **Revalidate**: `revalidatePath("/workspaces")`.

11. Return `{ success: true }`.

All error returns must be user-friendly Chinese strings (matching project i18n pattern).
