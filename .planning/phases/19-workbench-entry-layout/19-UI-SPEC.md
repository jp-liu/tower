## Spacing Scale

Standard 8-point scale. All values are multiples of 4.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps (`gap-1`), inline badge padding |
| sm | 8px | Compact element spacing (`gap-2`, `px-2`) |
| md | 16px | Default element/panel padding (`px-4`, `py-4`) |
| lg | 24px | Section padding (`px-6`) |
| xl | 32px | Layout gaps (not used in this phase) |
| 2xl | 48px | Major section breaks (not used in this phase) |
| 3xl | 64px | Page-level spacing (not used in this phase) |

**Exceptions:**
- Panel header height: `py-3` (12px top/bottom) — matches existing task-page-client.tsx header pattern.
- Back link / small controls: `py-0.5` (2px) — inline with text.
- Tab bar item: `px-4 py-3` (16px / 12px) — matches existing tab pattern in task-page-client.tsx.
- Resize handle: 4px width, no padding — visual divider only.

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px (`text-sm`) | 400 (regular) | 1.5 | Message content, placeholder text, tab body |
| Label | 12px (`text-xs`) | 400 (regular) | 1.4 | Back link, status badge text, branch badge, secondary metadata |
| Heading | 14px (`text-sm`) | 600 (semibold) | 1.2 | Task title in panel header, tab bar labels |
| Code/Mono | 12px (`text-xs font-mono`) | 400 (regular) | 1.5 | Branch name badge |

**Rule:** Exactly 2 weights in use: 400 (regular) and 600 (semibold). No other weights.

## Color

All values from `globals.css` CSS custom properties. Phase uses both light (Studio) and dark (Midnight Studio) themes via Tailwind dark mode.

| Role | Light value | Dark value | Usage |
|------|-------------|------------|-------|
| Dominant (60%) | `--background` oklch(0.97 0.005 260) | oklch(0.15 0.005 260) | Main page surface, right panel body |
| Secondary (30%) | `--sidebar` oklch(0.95 0.006 260) | oklch(0.13 0.006 260) | Left chat panel (`bg-sidebar`), panel headers |
| Accent (10%) | `--primary` oklch(0.65 0.14 75) | oklch(0.80 0.14 75) | Active tab indicator border, "查看详情" hover foreground |
| Destructive | `--destructive` oklch(0.55 0.2 25) | oklch(0.65 0.2 25) | Not used in this phase |
| Border | `--border` oklch(0.82 0.008 260) | oklch(0.28 0.008 260) | Panel dividers, tab bar border, resize handle |
| Muted text | `--muted-foreground` oklch(0.50 0.008 260) | oklch(0.65 0.008 260) | Back link, placeholder tab content |

**Accent reserved for:**
1. Active tab indicator: `border-b-2 border-primary` on the selected tab button.
2. "查看详情" button hover text color: `hover:text-foreground` (transitions from `text-muted-foreground`).
3. Resize handle hover state: `bg-primary/20` on drag hover.

**Accent NOT used for:** inactive tab labels, panel headers, placeholder copy, badge backgrounds.

**Status colors (pre-existing, carry forward unchanged):**

| Status | Class |

## Layout Contract

### Page Layout (`task-page-client.tsx` refactored)

```
┌─────────────────────────────────────────────────────────────┐
│  ResizablePanelGroup (horizontal, h-screen, bg-background)  │
│  ┌────────────────────┐ │ ┌───────────────────────────────┐ │
│  │  Left Panel (chat) │ │ │  Right Panel (tabs)           │ │
│  │  default: 35%      │ │ │  default: 65%                 │ │
│  │  min: 20%          │ │ │  min: 20%                     │ │
│  │  bg-sidebar        │ │ │  bg-background                │ │
│  │                    │ │ │  ┌─────────────────────────┐  │ │
│  │  [Header]          │ │ │  │  Tab Bar                │  │ │
│  │  [Conversation]    │ │ │  │  Files │ Changes │ Preview  │ │
│  │  [MessageInput]    │ │ │  └─────────────────────────┘  │ │
│  └────────────────────┘ │ │  [Tab Content Area]           │ │
│                         │ └───────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

**Library:** `react-resizable-panels@^2.x` (NOT v4 — v4 has breaking export renames). Source: CONTEXT.md D-01.

**Panel proportions:** Left 35 / Right 65 (defaultSize props). Source: CONTEXT.md D-02.

**Panel min width:** 20% each side (minSize={20}). Source: CONTEXT.md D-03.

**Resize handle appearance:** 4px wide, `bg-border` at rest, `bg-primary/20` on drag hover, cursor `col-resize`.

### Left Panel Header

```
┌─────────────────────────────────────────────────────┐
│ [← 返回]  [Task Title (truncate)]  [Status badge]   │
│            [Branch badge (font-mono)]               │
└─────────────────────────────────────────────────────┘
```

- Back link: `ArrowLeft` icon (h-3.5 w-3.5) + "返回" text, `text-xs text-muted-foreground hover:text-foreground`.
- Task title: `text-sm font-semibold text-foreground truncate`.
- Status badge: inline-flex, `text-[10px] font-medium`, per STATUS_COLORS table above.
- Branch badge: `Badge variant="secondary"`, `text-[10px] font-mono`, `GitBranch` icon (h-2.5 w-2.5).

### Right Panel Tab Bar

**Tab order:** Files → Changes → Preview. **Default selected:** Files. Source: CONTEXT.md D-04.

**Tab appearance:**
- Inactive: `px-4 py-3 text-sm font-medium text-muted-foreground hover:text-foreground`
- Active: `px-4 py-3 text-sm font-medium text-foreground border-b-2 border-primary`
- Each tab has icon + text label, icon on the left: `gap-2 flex items-center`

**Icons (lucide-react):** Source: CONTEXT.md D-05.

| Tab | Icon | Icon size |
|-----|------|-----------|
| Files | `FolderTree` | h-4 w-4 |
| Changes | `GitCompare` | h-4 w-4 |
| Preview | `Eye` | h-4 w-4 |

**Preview tab visibility:** Hidden (not rendered) when `project.type === "BACKEND"`. Uses conditional render, not CSS display:none. Source: CONTEXT.md D-06.

### "查看详情" Entry Button (in task-detail-panel.tsx)

**Placement:** Task drawer header, next to task title. Source: CONTEXT.md D-07.

**Appearance:** `button` element (not `Link`), `flex items-center gap-1 text-xs text-muted-foreground hover:text-foreground transition-colors`.

**Icon:** `ExternalLink` (h-3 w-3), leading icon.

**Label:** Uses i18n key `taskPage.viewDetails` → "查看详情" / "View Details" (already exists in i18n.tsx).

**Interaction:** `onClick` calls `router.push(\`/workspaces/\${workspaceId}/tasks/\${task.id}\`)`.

**Note:** The "查看详情" button may already exist in task-detail-panel.tsx (confirmed present at line 248-252). Verify before modifying — may only need to confirm it is wired to the correct route.

## Interaction States

### Tab Switching

- Tab state is local React state (`useState<"files" | "changes" | "preview">`), NOT URL-based.
- Switching tabs does NOT reset chat messages, scroll position, or diff data.
- The `TaskConversation` component remains mounted throughout (left panel never unmounts on tab switch).
- Default active tab on page load: `"files"`.

### Resize Handle

| State | Visual |
|-------|--------|
| At rest | 4px `bg-border` vertical bar |
| Hover | `bg-primary/20` |
| Active drag | `bg-primary/40`, `cursor-col-resize` |

### "查看详情" Button

| State | Visual |
|-------|--------|
| Default | `text-muted-foreground` |
| Hover | `text-foreground` with `transition-colors` |
| No active / focus / disabled states needed |
