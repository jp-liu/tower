  <action>
Create `src/components/assistant/assistant-provider.tsx` with "use client" directive.

The provider manages:
- `isOpen: boolean` — whether the assistant panel is visible
- `isStarting: boolean` — loading state while POST is in-flight (prevents terminal mount before session exists, per Pitfall 2 in RESEARCH.md)
- `displayMode: "sidebar" | "dialog"` — read from SystemConfig `assistant.displayMode` on mount, default "sidebar"
- `worktreePath: string | null` — set from POST response, cleared on close
- `toggleAssistant()` — if open, calls closeAssistant; if closed, calls openAssistant
- `closeAssistant()` — sets isOpen=false, then fires DELETE /api/internal/assistant, clears worktreePath

**openAssistant flow (critical ordering per Pitfall 2):**
1. Set `isStarting = true`
2. Call `fetch("/api/internal/assistant", { method: "POST" })`
3. Parse response JSON, extract `worktreePath`
4. Set `worktreePath` from response
5. Set `isOpen = true`, `isStarting = false`
6. On error: set `isStarting = false`, show `toast.error(t("assistant.errorBody"))` via sonner

**closeAssistant flow:**
1. Set `isOpen = false`, `worktreePath = null`
2. Fire `fetch("/api/internal/assistant", { method: "DELETE" })` (fire-and-forget, no await needed)

**Keyboard shortcut (per UI-02, UI-06):**
Register `useEffect` that listens for `keydown` on `document`:
- If `(e.metaKey || e.ctrlKey) && e.key === "l"` — call `e.preventDefault()` then `toggleAssistant()`
- This is registered at the provider level so it works from any page including full-screen task pages.

**Config read on mount:**
```typescript
useEffect(() => {
  getConfigValue<string>("assistant.displayMode", "sidebar").then((mode) => {
    setDisplayMode(mode === "dialog" ? "dialog" : "sidebar");
  });
}, []);
```

Export `AssistantProvider` component and `useAssistant()` hook.
`useAssistant()` throws if used outside provider.

Context value shape:
```typescript
interface AssistantContextValue {
  isOpen: boolean;
  isStarting: boolean;
  displayMode: "sidebar" | "dialog";
  worktreePath: string | null;
  toggleAssistant: () => void;
  closeAssistant: () => void;
}
```

Import `toast` from "sonner" for error handling. Import `getConfigValue` from "@/actions/config-actions". Import `useI18n` from "@/lib/i18n" for error message translation. Use `useCallback` for stable function references. Include `toggleAssistant` in the keyboard effect dependency array.
