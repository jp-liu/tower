**Resize useEffect** (separate effect, deps: `[]`, runs once):
Set up ResizeObserver on `containerRef.current`. Callback:
```typescript
const handler = debounce(() => {
  const fit = fitAddonRef.current;
  const ws = wsRef.current;
  const term = terminalRef.current;
  if (!fit || !term) return;
  fit.fit();
  if (ws?.readyState === WebSocket.OPEN) {
    ws.send(JSON.stringify({ type: "resize", cols: term.cols, rows: term.rows }));
  }
}, 100);   // D-06: 100ms debounce
```
Use a local `debounce` implementation (no lodash dep):
```typescript
function debounce<T extends (...args: unknown[]) => void>(fn: T, ms: number): T {
  let timer: ReturnType<typeof setTimeout> | null = null;
  return ((...args) => {
    if (timer) clearTimeout(timer);
    timer = setTimeout(() => fn(...args), ms);
  }) as T;
}
```
Return cleanup: `resizeObserver.disconnect()`.
