<objective>
Strip UUID suffixes from cache filenames when copying reference files to project assets during task creation, restoring human-readable names.

Purpose: Cache files include an 8-hex UUID suffix for uniqueness (e.g. `设计稿-a1b2c3d4.png`). When these become permanent project assets, the UUID is noise. Stripping it produces clean, readable asset names.

Output: Two new exported helpers in file-utils.ts, wired into the task-tools.ts reference copy loop, with full test coverage.
</objective>

const CACHE_UUID_SUFFIX_RE = /-([0-9a-f]{8})(\.[^.]+)$/i;

}
