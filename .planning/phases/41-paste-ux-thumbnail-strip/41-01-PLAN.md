Create `src/hooks/use-image-upload.ts` with the following:

1. Export `PendingImage` interface:
```typescript
export interface PendingImage {
  id: string;
  file: File;
  blobUrl: string;
  status: "uploading" | "done" | "error";
  progress: number; // 0-100
  filename?: string; // populated after successful upload
}
```

2. Export `useImageUpload()` hook returning `{ pendingImages, addImages, removeImage, clearAll, hasUploading }`:

- `addImages(files: File[])`: For each file, generate `id` via `crypto.randomUUID()`, create `blobUrl` via `URL.createObjectURL(file)`, add to state with status "uploading" and progress 0. Then start XHR upload:
  - Store XHR instance in `useRef<Map<string, XMLHttpRequest>>()` keyed by image id
  - `xhr.upload.onprogress`: update progress via `Math.round((e.loaded / e.total) * 100)` using immutable state update (map over prev)
  - `xhr.onload` (NOT `xhr.upload.onload`): if status 200, parse `JSON.parse(xhr.responseText)` to get `{ filename }`, set status "done" + progress 100 + filename. Otherwise set status "error"
  - `xhr.onerror`: set status "error"
  - Delete from xhrMap on load/error
  - `xhr.open("POST", "/api/internal/assistant/images")`, send `new FormData()` with `formData.append("file", file)`

- `removeImage(id: string)`: Get XHR from map, abort if exists, delete from map. Update state: find image, call `URL.revokeObjectURL(img.blobUrl)`, filter it out.

- `clearAll()`: Iterate all pending images — abort XHRs from map, revoke all blob URLs, clear map, set state to empty array.

- `hasUploading`: derived `pendingImages.some((i) => i.status === "uploading")`

- Cleanup `useEffect` (empty deps): On unmount, use a `imagesRef` (useRef mirroring pendingImages) to revoke all blob URLs and abort all XHRs. Do NOT call setState inside the cleanup function (React 19 warning). Pattern:
```typescript
const imagesRef = useRef<PendingImage[]>([]);
useEffect(() => { imagesRef.current = pendingImages; }, [pendingImages]);
useEffect(() => {
  return () => {
    imagesRef.current.forEach((img) => URL.revokeObjectURL(img.blobUrl));
    xhrMap.current.forEach((xhr) => xhr.abort());
    xhrMap.current.clear();
  };
}, []);
```

All state updates must be immutable (spread/map/filter). No `console.log` in production code.
