**1. Create `src/lib/diff-parser.ts`** — Pure utility, no Next.js imports.

Export these types and functions:

```typescript
export interface DiffFile {
  filename: string;
  added: number;
  removed: number;
  isBinary: boolean;
  patch: string;      // unified diff content for this file
}

export interface DiffResponse {
  files: DiffFile[];
  totalAdded: number;
  totalRemoved: number;
  hasConflicts: boolean;
  conflictFiles: string[];
}
```

`export function parseDiffOutput(numstat: string, unifiedDiff: string): Omit<DiffResponse, "hasConflicts" | "conflictFiles">`:
- Parse numstat: each line is `{added}\t{removed}\t{filename}`. Lines with `-\t-` are binary files (set isBinary=true, added=0, removed=0).
- Parse unified diff: split on lines matching `^diff --git a/.+ b/.+` regex. Each segment after the header is the patch for that file. Map filename from the `diff --git a/{name} b/{name}` header.
- Match numstat entries to unified diff segments by filename.
- Compute totalAdded/totalRemoved as sums.
- If unified diff output exceeds 500KB in total length, truncate individual file patches to first 200 lines and append `\n... (truncated)`.

`export function checkConflicts(localPath: string, baseBranch: string, worktreeBranch: string): { hasConflicts: boolean; conflictFiles: string[] }`:
- Run `git merge-tree --write-tree ${baseBranch} ${worktreeBranch}` with `execSync`, cwd=localPath, timeout=10000, stdio="pipe".
- If exit code is 0: return `{ hasConflicts: false, conflictFiles: [] }`.
- If the command throws (exit code 1): parse stdout for lines containing `CONFLICT` to extract filenames. Return `{ hasConflicts: true, conflictFiles: [...] }`.
- Import `execSync` from `child_process`.

import { NextRequest, NextResponse } from "next/server";

GET handler:
- Extract taskId from params: `const { taskId } = await params;`
- Load task with project: `db.task.findUnique({ where: { id: taskId }, include: { project: true } })`.
- Guard: if no task, return 404. If `!task.baseBranch`, return 400 `{ error: "Task has no base branch configured" }`. If `!task.project?.localPath`, return 400 `{ error: "Project has no local path" }`.
- Get latest COMPLETED execution: `db.taskExecution.findFirst({ where: { taskId, status: "COMPLETED" }, orderBy: { createdAt: "desc" } })`.
- Derive `worktreeBranch` from execution's `worktreeBranch` field, fallback to `task/${taskId}`.
- Set `localPath = task.project.localPath`.
- Run numstat: `execSync(\`git diff --numstat ${task.baseBranch}...${worktreeBranch}\`, { cwd: localPath, encoding: "utf-8", timeout: 30000 })`.
- Run unified: `execSync(\`git diff --unified=3 ${task.baseBranch}...${worktreeBranch}\`, { cwd: localPath, encoding: "utf-8", timeout: 30000 })`.
- Parse with `parseDiffOutput(numstat, unified)`.
- Check conflicts with `checkConflicts(localPath, task.baseBranch, worktreeBranch)`.
- Also get commit count: `execSync(\`git rev-list --count ${task.baseBranch}...${worktreeBranch}\`, { cwd: localPath, encoding: "utf-8", timeout: 5000 }).trim()`.
- Return `NextResponse.json({ ...parsedDiff, hasConflicts, conflictFiles, commitCount: parseInt(commitCount, 10) || 0 })`.
- Wrap in try/catch: on error return 500 with `{ error: "Failed to generate diff" }`.

**1. Create `src/app/api/tasks/[taskId]/merge/route.ts`** — POST handler.

```typescript
import { NextRequest, NextResponse } from "next/server";
import { db } from "@/lib/db";
import { execSync } from "child_process";
import { revalidatePath } from "next/cache";
import { checkConflicts } from "@/lib/diff-parser";
```

POST handler (per D-07 through D-11):
- Extract taskId from params.
- Load task with project: `db.task.findUnique({ where: { id: taskId }, include: { project: true } })`.
- Guards: 404 if no task. 400 if `task.status !== "IN_REVIEW"` with error `"Task must be in IN_REVIEW status to merge"`. 400 if `!task.baseBranch` with error `"Task has no base branch configured"`. 400 if `!task.project?.localPath`.
- Get latest COMPLETED execution for worktreeBranch: `db.taskExecution.findFirst({ where: { taskId, status: "COMPLETED" }, orderBy: { createdAt: "desc" } })`.
- Derive `worktreeBranch` from execution or fallback to `task/${taskId}`.
- Set `localPath = task.project.localPath`.
- **Pre-merge conflict check** (per D-10): call `checkConflicts(localPath, task.baseBranch, worktreeBranch)`. If `hasConflicts`, return 409 `{ error: "Merge conflicts detected", conflictFiles }`.
- **Squash merge** (per D-09): run these three commands sequentially in the main repo (cwd = localPath, NOT worktree):
  1. `execSync(\`git checkout ${task.baseBranch}\`, { cwd: localPath, encoding: "utf-8", timeout: 10000 })`
  2. `execSync(\`git merge --squash ${worktreeBranch}\`, { cwd: localPath, encoding: "utf-8", timeout: 30000 })`
  3. `execSync(\`git commit -m "feat: ${task.title.replace(/"/g, '\\"')}"\`, { cwd: localPath, encoding: "utf-8", timeout: 10000 })`
- **Update status to DONE** (per D-11): `await db.task.update({ where: { id: taskId }, data: { status: "DONE" } })`.
- Call `revalidatePath("/workspaces")`.
- Return `NextResponse.json({ success: true, message: "Squash merge completed" })`.
- Wrap in try/catch: on git error return 500 with `{ error: "Merge failed", details: err.message }`.
