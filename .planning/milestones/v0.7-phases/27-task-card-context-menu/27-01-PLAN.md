Props interface:
```typescript
interface TaskCardContextMenuProps {
  x: number;
  y: number;
  taskId: string;
  currentStatus: TaskStatus;
  hasExecutions: boolean;           // true → disable launch item
  workspaceId: string;
  onClose: () => void;
  onStatusChange: (taskId: string, status: TaskStatus) => void;
  onLaunch: (taskId: string) => void;
}
```

Implementation requirements:
1. Dismiss on outside mousedown and Escape key (same useEffect pattern as FileTreeContextMenu).
2. Render via createPortal to document.body. Guard with `if (typeof document === "undefined") return null`.
3. Position: `style={{ position: "fixed", top: y, left: x, zIndex: 9999 }}`.
4. Menu container className: `"bg-popover border border-border rounded-md shadow-md py-1 min-w-[180px]"`.
5. Standard item class constant: `"flex items-center px-3 py-1.5 text-xs cursor-pointer hover:bg-accent rounded-sm w-full text-left"`.

Menu structure (top to bottom):
- Section label (non-interactive): `t("board.contextMenu.changeStatus")` — small muted text, px-3 py-1 text-[10px] text-muted-foreground uppercase tracking-wider
- One `<button>` per BOARD_COLUMNS entry (import from "@/lib/constants"): shows column label; current status gets a checkmark icon (lucide Check, h-3 w-3 mr-2); others get empty spacer (w-5); clicking calls `onStatusChange(taskId, col.id)` then `onClose()`.
- `<hr className="my-1 border-border" />` separator
- Launch button: `t("board.contextMenu.launch")` — disabled + `opacity-50 cursor-not-allowed` when `hasExecutions` is true. When enabled, clicking calls `onLaunch(taskId)` then `onClose()`.
- `<hr className="my-1 border-border" />` separator
- Detail button: `t("board.contextMenu.goToDetail")` — import `useRouter` from "next/navigation"; clicking calls `router.push(\`/workspaces/${workspaceId}/tasks/${taskId}\`)` then `onClose()`.

Use `useI18n` for all label strings (per D-01/D-02/D-03 from CONTEXT.md). Import `TaskStatus` from "@prisma/client". Import `BOARD_COLUMNS` from "@/lib/constants". Import `Check` from "lucide-react".
