## Spacing Scale

Standard 8-point scale. All values are multiples of 4.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps (`gap-1`), inline badge padding |
| sm | 8px | Compact element spacing (`gap-2`, `px-2`) |
| md | 16px | Default element/panel padding (`px-4`, `py-4`) |
| lg | 24px | Section padding (`px-6`) |
| xl | 32px | Layout gaps (not used in this phase) |
| 2xl | 48px | Major section breaks (not used in this phase) |
| 3xl | 64px | Page-level spacing (not used in this phase) |

**Exceptions:**
- Panel header height: `py-3` (12px top/bottom) вҖ” matches existing task-page-client.tsx header pattern.
- Back link / small controls: `py-1` (4px) вҖ” inline with text.
- Tab bar item: `px-4 py-3` (16px / 12px) вҖ” matches existing tab pattern in task-page-client.tsx.
- Resize handle: 4px width, no padding вҖ” visual divider only.

---

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px (`text-sm`) | 400 (regular) | 1.5 | Message content, placeholder text, tab body |
| Label | 12px (`text-xs`) | 400 (regular) | 1.4 | Back link, branch badge, secondary metadata |
| Heading | 14px (`text-sm`) | 600 (semibold) | 1.2 | Task title in panel header, active tab labels, status badge text |
| Code/Mono | 12px (`text-xs font-mono`) | 400 (regular) | 1.5 | Branch name badge |

**Rule:** Exactly 2 weights in use: 400 (regular) and 600 (semibold). No other weights.

## Color

All values from `globals.css` CSS custom properties. Phase uses both light (Studio) and dark (Midnight Studio) themes via Tailwind dark mode.

| Role | Light value | Dark value | Usage |
|------|-------------|------------|-------|
| Dominant (60%) | `--background` oklch(0.97 0.005 260) | oklch(0.15 0.005 260) | Main page surface, right panel body |
| Secondary (30%) | `--sidebar` oklch(0.95 0.006 260) | oklch(0.13 0.006 260) | Left chat panel (`bg-sidebar`), panel headers |
| Accent (10%) | `--primary` oklch(0.65 0.14 75) | oklch(0.80 0.14 75) | Active tab indicator border, "жҹҘзңӢиҜҰжғ…" hover foreground |
| Destructive | `--destructive` oklch(0.55 0.2 25) | oklch(0.65 0.2 25) | Not used in this phase |
| Border | `--border` oklch(0.82 0.008 260) | oklch(0.28 0.008 260) | Panel dividers, tab bar border, resize handle |
| Muted text | `--muted-foreground` oklch(0.50 0.008 260) | oklch(0.65 0.008 260) | Back link, placeholder tab content, inactive tab labels |

**Accent reserved for:**
1. Active tab indicator: `border-b-2 border-primary` on the selected tab button.
2. "жҹҘзңӢиҜҰжғ…" button hover text color: `hover:text-foreground` (transitions from `text-muted-foreground`).
3. Resize handle hover state: `bg-primary/20` on drag hover.

**Accent NOT used for:** inactive tab labels, panel headers, placeholder copy, badge backgrounds.

**Status colors (pre-existing, carry forward unchanged):**

| Status | Class |
|--------|-------|
| TODO | `bg-muted text-muted-foreground` |
| IN_PROGRESS | `bg-blue-500/10 text-blue-400 border border-blue-500/20` |
| IN_REVIEW | `bg-amber-500/10 text-amber-400 border border-amber-500/20` |
