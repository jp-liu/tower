## Advisory Notes

1. **Image serving uses localhost-only routes** — consistent with app's local-only design. Remote access would break image display.
2. **MSG-04 partial** — session image persistence uses `sessionStorage` (tab-scoped). F5 reload works; tab close+reopen loses images. This is a known limitation of the file-path SDK approach.
3. **CACHE-05 orphaned** — assets route is valid infrastructure for future use by MCP tools or other features.
